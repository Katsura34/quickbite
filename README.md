# QuickBite - Local Food Ordering System

A simple, browser-based food ordering system that works entirely offline using localStorage for data persistence.

## Features

### Customer Features
- Browse food menu with category filtering
- Add items to cart
- Place orders
- View order history

### Admin Features
- Order management (view pending and completed orders)
- Menu management (add, edit, delete menu items)
- Sales reports with date filtering

## Getting Started

Simply open `index.html` in any modern web browser. No server setup required!

### Default Accounts

**Admin Account:**
- Username: `admin`
- Password: `admin123`

**Customer Account:**
- Username: `customer`
- Password: `customer123`

Or create a new account using the registration form.

## How It Works

All data is stored in the browser's localStorage:
- User accounts
- Menu items
- Shopping cart
- Order history

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- localStorage for data persistence
